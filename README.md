# Darmi - Technical Support

Welcome to the official support page for **Darmi**.

A simple, calm way to track IBS triggers.

Living with IBS means second-guessing every meal.
Was it the onions? Stress? Coffee? Nothing makes sense.

This app helps you:
• Log food quickly
• Record symptoms and intensity
• Track mood and stress
• View trends over time
• Notice patterns without overthinking

This is not medical advice.
It’s a private space to understand your own body.
Built by someone who needed it.
